La aplicación de control robertiño servirá únicamente como retroalimentador al docente, que utilizará el ecosistema educativo, por lo que si se desea utilizar su código fuente, deberá ajustarse su código a las características del sistema operativo donde se quiere ejecutar.

Secciones como por ejemplo:

**my_application.cc**: secciones como `ejecutarAPI` y `extraerIP` deben modificarse, puesto que cada distribución de Linux funciona de forma diferente.

El código fuente debe ser modificado y adaptado a cada entorno de ejecución.
